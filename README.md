# 📚 Public Library Management System

## 📖 Overview
The **Public Library Management System** is a web-based application designed to efficiently manage and track library books. It helps maintain records of available and issued books while simplifying overall library operations through a structured and user-friendly system.

---

## ✨ Features
- 📘 Display all available books in the library
- 📕 Track issued books separately
- 📂 Maintain records of book availability
- 🔍 Advanced book search and filter functionality
- 👤 User authentication and login system
- 🛡️ Role-based access control for Admin and Users
- 📊 Dashboard analytics for issued and available books
- ⚡ Easy and efficient library inventory management

---

## ⚙️ Functionality
- ➕ Add new books to the library
- 📤 Issue books to users
- 📥 Return issued books back to inventory
- 📑 View available and issued books records
- 🔐 Secure login and authentication system
- 🗄️ Store and manage records using MySQL database

---

## 🎯 Purpose
The main purpose of this project is to simplify library management operations and maintain proper book records in an organized, efficient, and secure way.
